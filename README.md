# Assignment_Kithokoi

This repository contains my LuxDev SQL assignment. It includes the SQL script used to create the database schema, define the tables, insert sample data, and answer the assigned SQL questions.

## Project Overview

In this assignment, I created a schema and built the following tables:

- `customers`
- `products`
- `sales`
- `inventory`

After creating the tables, I inserted data and then solved a series of 50 assigned SQL questions based on the dataset.

## File Included

- `assignment.sql` — contains the schema creation, table creation, data insertion, and SQL solutions for the assignment questions.

## What This Assignment Covers

This project demonstrates practical SQL skills, including:

- creating a schema
- creating tables
- inserting data into tables
- querying relational data
- solving SQL assignment questions

## Purpose

The purpose of this assignment is to show understanding of core SQL concepts and how they are applied in a structured database environment.
